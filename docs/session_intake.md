# 新規セッション希望ヒアリング

新規開始時、シナリオ考案前に以下 3 軸を PL に確認する。各軸「おまかせ」可。回答はシナリオ設計・判定運用・セッション尺に反映する。

## ジャンル
1. 現代ミステリ
2. 歴史・時代劇
3. ファンタジー
4. SF
5. ホラー
6. 日常・コメディ
7. サイバーパンク
8. ポストアポカリプス
9. おまかせ
10. その他（自由記述）— 上記にないジャンル・複合ジャンル・具体的な作品名を参照指定も可。例：「魔法学園もの」「ノワール × 海洋冒険」「〇〇風の世界観で」

## 難易度
1. **軽め** — 致命的判定ほぼなし、失敗でも物語が転ぶ程度。§4.4 能力裁定は緩め、§4.2 判定頻度も最小
2. **標準** — 戦闘・重要判定あり、PC 死亡の可能性は低め
3. **高難度** — 序盤から危機、能力適用の裁定も厳しめ、PC 死亡あり得る

## 尺
1. **短編** — 1〜3 シーン、1 セッション完結
2. **中編** — 5〜8 シーン、セーブ 1〜2 回想定
3. **長編** — 章立て、セーブ複数前提の読み応え重視
4. **キャンペーン** — 複数シナリオ連結、PC 継続・成長描写あり（`docs/campaign.md` 参照）
